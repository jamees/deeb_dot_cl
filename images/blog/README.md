## **Herramientas de Optimización**

Antes de subir, optimiza las imágenes:

### **Online:**
- [TinyPNG](https://tinypng.com/) - Compresión sin pérdida
- [Squoosh](https://squoosh.app/) - Optimización avanzada
- [ImageOptim](https://imageoptim.com/online) - Batch optimization

### **CLI:**
```bash
# Instalar ImageMagick
brew install imagemagick

# Optimizar y redimensionar
magick convert input.jpg -resize 800x600^ -quality 85 output.jpg
```
